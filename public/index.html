vercel-demo

This site is a minimal demo deployed to Vercel. The API endpoints are available under /api/*.

  • /api/health — health check (should return JSON)
  • /api/todos — sample TODO API (GET/POST)

If you see this page, the deployment's static routing is working. If you need a different root behavior (SPA or redirect), update vercel.json or add an index.html in the desired location.
